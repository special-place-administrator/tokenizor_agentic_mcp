  <files>src/live_index/store.rs, src/watcher/mod.rs, src/lib.rs, Cargo.toml</files>
  <behavior>
    - update_file("src/foo.rs", indexed_file) inserts or replaces the entry and updates loaded_at_system
    - add_file("src/new.rs", indexed_file) inserts a new entry (alias for update_file semantics)
    - remove_file("src/deleted.rs") removes the entry and updates loaded_at_system; returns silently if path not present
    - remove_file on non-existent path does NOT update loaded_at_system
    - After update_file, get_file returns the new IndexedFile
    - After remove_file, get_file returns None
    - After add_file on a fresh index, file_count increases by 1
    - WatcherState::Active, WatcherState::Degraded, WatcherState::Off are distinct variants
    - WatcherInfo default: state=Off, events_processed=0, last_event_at=None
  </behavior>
  <action>
    1. Add to Cargo.toml [dependencies]:
       ```

       notify-debouncer-full = "0.7"
       ```
       Also add `tokio` feature "time" (needed for watcher backoff sleep) — update the existing tokio line to include "time" in features.

    2. Create `src/watcher/mod.rs` with type definitions only (no watcher logic yet):
       - `pub enum WatcherState { Active, Degraded, Off }` — derive Clone, Debug, PartialEq, Eq
       - `pub struct WatcherInfo { pub state: WatcherState, pub events_processed: u64, pub last_event_at: Option<SystemTime>, pub debounce_window_ms: u64 }` — derive Clone, Debug
       - `impl Default for WatcherInfo` returning Off state, 0 events, None last_event, 200ms window
       - `pub struct BurstTracker { pub event_count: u32, pub window_start: Instant, pub last_event_at: Instant, pub extended: bool }`
       - `impl BurstTracker` with `pub fn new() -> Self` and `pub fn update(&mut self, now: Instant)` and `pub fn effective_debounce_ms(&self) -> u64`
       - BurstTracker constants: BURST_THRESHOLD=3, BASE_MS=200, BURST_MS=500, QUIET_SECS=5
       - BurstTracker::update: if now - window_start > 200ms, reset window. Else increment count. If count > 3, set extended=true. Always update last_event_at.
       - BurstTracker::effective_debounce_ms: if last_event_at.elapsed() > 5s, return 200 (reset). If extended, return 500. Else return 200.

    3. Add `pub mod watcher;` to `src/lib.rs`.

    4. Add three methods to `impl LiveIndex` in `src/live_index/store.rs`:
       ```rust
       pub fn update_file(&mut self, path: String, file: IndexedFile) {
           self.files.insert(path, file);
           self.loaded_at_system = SystemTime::now();
       }

       pub fn add_file(&mut self, path: String, file: IndexedFile) {
           self.update_file(path, file);
       }

       pub fn remove_file(&mut self, path: &str) {
           if self.files.remove(path).is_some() {
               self.loaded_at_system = SystemTime::now();
           }
       }
       ```

    5. Write unit tests in `src/live_index/store.rs` (inside existing `#[cfg(test)] mod tests` if present, or create one):

       - test_remove_file_removes_existing
       - test_remove_file_nonexistent_is_noop (loaded_at_system should NOT change)
       - test_file_count_after_mutations

    6. Write unit tests in `src/watcher/mod.rs`:
       - test_watcher_state_variants (all 3 exist and are distinct)
       - test_watcher_info_default (Off, 0, None, 200)
       - test_burst_tracker_new (initial state: count=0, extended=false)
       - test_burst_tracker_extends_window (4 events within 200ms -> extended=true, effective=500)
       - test_burst_tracker_resets_after_quiet (last_event > 5s ago -> effective=200)
       - test_burst_tracker_new_window_resets_count (event after 200ms gap -> count=1, extended=false)
