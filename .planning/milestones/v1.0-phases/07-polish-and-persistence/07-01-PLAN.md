   - `pub fn extract_symbols(node: &Node, source: &str) -> Vec<SymbolRecord>`
   - `walk_node` recursive function recognizing: `function_definition` -> Function, `struct_specifier` -> Struct, `enum_specifier` -> Enum, `type_definition` -> Type
   - `find_c_name` helper: For `function_definition`, walk `declarator` child recursively to find `function_declarator` then extract first `identifier`. For `struct_specifier`/`enum_specifier`, find child with kind `type_identifier`. For `type_definition`, find inner specifier or `type_identifier`.

   - For C++ name extraction: `function_declarator` may contain `qualified_identifier` instead of plain `identifier` — handle both. Extract the final segment (after `::`) as the symbol name.
   - `template_declaration`: find the inner `function_definition`, `class_specifier`, or `struct_specifier` child and recurse into it.

4. Wire into `src/parsing/languages/mod.rs`:
   - Add `mod c;` and `mod cpp;`
   - Add match arms: `LanguageId::C => c::extract_symbols(node, source)`, `LanguageId::Cpp => cpp::extract_symbols(node, source)`

   ```
   (call_expression function: (field_expression field: (field_identifier) @ref.method_call))
   ; #include imports
   (preproc_include path: (string_literal) @ref.import)
   ```

2. Add C++ xref query (extends C patterns):
   ```
   const CPP_XREF_QUERY: &str = r#"
   ; Function calls (same as C)
   (call_expression function: (identifier) @ref.call)
   (call_expression function: (field_expression field: (field_identifier) @ref.method_call))
   ; Qualified calls: std::sort, Foo::bar
   (qualified_identifier name: (identifier) @ref.call)
   ; Includes
   (preproc_include path: (string_literal) @ref.import)
   (preproc_include path: (system_lib_string) @ref.import)
   ; Type identifiers
   (type_identifier) @ref.type
   ; Template instantiation: vector<int>
   (template_type name: (type_identifier) @ref.type)
   ; Using declarations: using std::string
   (using_declaration (qualified_identifier) @import.original)
   "#;
   ```
